Grammar · Week 8

Demonstra­tives

Vivumishi Viashiria — pointing to what's near, far, or already known

What are demonstratives?

A demonstrative is a pointing word — it identifies exactly which thing you mean. In English the demonstratives are this, these, that, those. Swahili has all of these, plus a special fourth type: the reference demonstrative, used when the thing has already been mentioned in the conversation.

Like adjectives, every demonstrative in Swahili must agree with the noun it points to — it changes shape based on the noun's class. But each type follows a clear, logical pattern that you can learn once and apply everywhere.

Demonstratives can work as adjectives (next to a noun: kitabu hiki — this book) or as pronouns (standing alone: Hiki ni kizuri — this one is good). The form is exactly the same either way.

Section A
The four types
Swahili has four demonstrative types. Two are based on physical distance (near vs. far). Two are reference types for things already discussed.
Proximity
this / these

Near the speaker

Always starts with H. Think: "H for Here." Used for things close to you right now.

Ref. Proximity
this/these (ref)

Mentioned, nearby

Built from proximity + a relative particle. Used when the nearby thing has already come up in conversation.

Distance
that / those

Far from the speaker

Always ends in -le. Think: "-le" = pointing away. Used for things away from you.

Ref. Distance
that/those (ref)

Mentioned, far away

Same form as distance. What changes is word order — it usually precedes the noun when no adjective is present.

Section B
How they are formed
Each type has its own formation rule. Learn the pattern once — it applies to every noun class.

Proximity — "this / these"

Every proximity demonstrative starts with h-, followed by a vowel that duplicates the vowel of the noun class's subject prefix, then the subject prefix itself.

WA- class: H + a + wa = hawa  |  MI- class: H + i + i = hii  |  JI- class: H + i + li = hili

Exception — M-WA singular: We would expect a- (the subject prefix) but it uses -yu instead, giving H + u + yu = huyu. Also: M(W)- locative has no vowel in its prefix, so the vowel u is inserted → humu.

Reference for Distance

No new forms needed! The Reference for Distance uses exactly the same forms as the Distance demonstrative. The only difference is word order — the reference form usually comes before the noun (when no adjective is present). See Section D for full word order rules.

Easy rule to remember: If you already know the distance forms (wale, yule, kile, lile…), you already know the reference-for-distance forms. Just learn when to place them before vs. after the noun.

Locative classes: PA- = a definite/specific spot (hapa = right here). KU- = a general direction or area (huku = around this way). M(W)- = the interior of a place (humu = inside here). The M(W)- proximity form is special — its prefix has no vowel, so u is inserted to make humu.
Section D
Word order rules
Where a demonstrative sits in a sentence depends on its type, and whether adjectives or possessives are also present.
Proximity and Distance — basic position
Normal everyday usage
Demonstrative comes AFTER the noun
nyumba hii
this house
watu wale
those people
Emphatic — strong pointing
Demonstrative comes BEFORE the noun for emphasis
Hii nyumba!
THIS house! (pointing firmly)
Wale watu!
THOSE people! (emphatic)
Reference for Proximity — three rules
Rule 1 — noun alone (no adjective)
Reference demonstrative comes BEFORE the noun
Hilo ua ni zuri.
This (referred to) flower is nice.
Hayo maua ni mazuri.
These (referred to) flowers are nice.
Huyo paka alilala.
This (referred to) cat was asleep.

Key contrast to remember: Both reference types follow the exact same three word order rules. What changes is only the demonstrative form — hilo/hayo for reference proximity vs. lile/yale for reference distance.

Sentences in action
All four types used in real sentences across different noun classes.
Proximity this / these
Ua hili ni zuri.This flower is nice.
Paka huyu alilala.This cat was asleep.
Vitabu hivi ni vizuri.These books are good.
Nyumba hii ni kubwa.This house is big.
Mahali pazuri hapa ni kasri la rais.This beautiful place is the presidential palace.
Kusoma kurefu huku kunapendeza.This long reading is lovely.
Ref. Proximity this/these (referred to)
Hilo ua ni zuri.This (referred to) flower is nice.
Huyo paka alilala.This (referred to) cat was asleep.
Ua hilo jeupe ni zuri.This (referred to) white flower is nice.
Ua langu hilo jeupe ni zuri.This (referred to) white flower of mine is nice.
Hicho chakula cha jamaa kinanukia.This (referred to) food of the relative smells good.
Hizo kweli ni wazi.These (referred to) truths are self-evident.
Distance that / those
Ua lile ni zuri.That flower is nice.
Paka yule alilala.That cat was asleep.
Vitabu vile ni ghali.Those books are expensive.
Simba kijana mnene yule amelala.That young fat lion has been asleep.
Mahali pazuri pale ni kasri la rais.That beautiful place is the presidential palace.
Kule kuimba ni kubaya.That (referred to) singing is bad.
