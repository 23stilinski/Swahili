Chapter 11 · Swahili Grammar

Object Infixes

How to embed "me / you / him / them" directly inside the verb.

Subject + Tense + Object Infix ← NEW + Verb Stem
§ 1 — The Big Idea

What Is an Object Infix?

You already know the verb formula: Subject + Tense + Verb.

An Object Infix is a tiny extra piece that slots in between the tense marker and the verb. It replaces or reinforces the object — the person or thing the action is being done to.

Think of it like this: instead of saying "I see him" you build "him" right into the verb → Nitamwona (I will see him).

Object Infixes are most common when referring to living things (people, animals) as the object, or when you want to emphasise the object.
eg.
Ataniona
He will see me.
A = he/she ta = future ni = me (object infix) ona = see
eg.
Ninakupenda
I love you.
Ni = I na = present ku = you (object infix) penda = love
§ 2 — People & Animals

The M-/WA- Class Infixes

For people (and animals/insects — they always follow this class), use these object infixes:

Who?Object InfixExample VerbMeaning
me-ni-A·ta·ni·saidiaHe/She will help me
you (sing.)-ku-Ni·na·ku·pendaI love you
him / her-m- / -mw-Ni·ta·mw·onaI will see him/her
us-tu-Wa·na·tu·saidiaThey are helping us
you (pl.)-wa-Ni·li·wa·ona·niI saw you all
them-wa-Ni·li·wa·onaI saw them
-m- vs -mw-: Use -mw- when the verb starts with a vowel → Nitamwona (see). Use -m- when the verb starts with a consonant → Nitamjibu (answer).
"you (pl.)" vs "them": Both use -wa-. To show "you all" specifically, add -ni to the end of the verb: Niliwaonani = I saw you all.

Nitamwona

Ni=Ita=futuremw=him/herona=see
03

They are helping us.
Wa=theyna=presenttu=ussaidia=help
04
Alimshukuru daktari
He/She thanked the doctor. (Object infix + noun reinforces it)

§ 3 — Things & Objects

Other Noun Classes — Things

For things (not people), the object infix is the same as the agreement subject prefix for that class. The key ones to know:

Noun ClassSingular InfixPlural InfixExample (Sing.)Example (Pl.)
M-/MI- (trees, things) -u--i- Tuliula mkate
We ate the bread		 Tuliila mikate
We ate the breads
JI-/MA- (fruits, big things) -li--ya- Ulilifungua sanduku
You opened the box		 Uliyafungua masanduku
You opened the boxes
KI-/VI- (chairs, things) -ki--vi- Nimekivunja kiti
I have broken the chair		 Nimevivunja viti
I have broken the chairs
N-/N- (animals, things) -i--zi- Atainunua baiskeli
He will buy the bicycle		 Atazinunua baiskeli
He will buy the bicycles
Notice the pattern: singular infix → plural infix just swaps one letter, e.g. -ki- → -vi-, -li- → -ya-. Learn the pair, not each one separately.

§ 4 — Must-Know Rule

3 Verbs That Always Take an Object Infix

Three verbs in Swahili must always have an object infix — and it must always be from the M-/WA- class, even if the object is a thing.

ambia to tell
Alimwambia kila kitu.
He told him/her everything.
ita to call
Tumewaita jirani.
We have called the neighbours.
pa to give
Nilikupa mkate.
I gave you bread.
Memory trick: AIP — Ambia, Ita, Pa. These three always need the M-/WA- object infix.
§ 5 — Extra Rule

When There Are Two Objects

A Swahili verb can only hold one object infix. So when a sentence has two objects, here's the rule:

👤 If one object is a living thing → it gets the infix; the other object is just the noun.

📦 If both objects are things → the direct object provides the infix.

eg.
Nitamsimulia hadithi
I will tell him/her a story. (him = infix; story = plain noun)
Ni=Ita=futurem=him/hersimulia=tell a storyhadithi=story
eg.
Nilikiona kitabu mfukoni
I saw the book in the bag. (book = direct object → KI- infix)
Ni=Ili=pastki=it (book)ona=seemfukoni=in bag
Monosyllabic verbs: When using an object infix with a short verb like -la (eat), -nywa (drink), drop the infinitive ku-. e.g. Niliyanywa maji → I drank the water.
