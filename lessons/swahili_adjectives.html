Vivumishi —
Adjectives 🌈

Words that describe nouns — and how they change to match!

🤔 What is an adjective?

An adjective is a word that describes a noun. In English: big, red, new, good. In Swahili they work the same way — but with one big difference.

In Swahili, the adjective has to "dress up" to match the noun it describes. If the noun wears an M- hat, the adjective wears one too. This is called agreement. Think of it like wearing matching outfits! 👗👔

There are 5 types of adjectives in Swahili. We'll go through each one simply and clearly.

Section A
Vowel Stem Adjectives
Adjectives whose stem starts with a vowel (a, e, i, o, u)
💡
The Big Idea
Some adjectives start with a vowel — like -ekundu (red) or -ingine (other/another).

When these adjectives describe a noun, you attach a prefix to the front — and the prefix + vowel sometimes blend together to make a new sound.

Think of it like adding chocolate sauce to ice cream — sometimes they just mix together smoothly! 🍫🍦
🔤 Vowels in Swahili: a, e, i, o, u — just like English. The letter Y is NOT a vowel in Swahili. So -yema would be a consonant stem, not vowel stem.
Let's start with the easiest noun classes first 👇
📚
KI / VI Class — Objects
KI class uses ch- before vowel stems.
VI class uses vy- before vowel stems.

-ekundu (red) → ki + ekundu = chekundu | vi + ekundu = vyekundu
-ingine (other) → ki + ingine = kingine | vi + ingine = vingine
🐘
N Class — Animals & Common Nouns
N class uses ny- before vowel stems.

-ekundu (red) → ny + ekundu = nyekundu
-ingine (other) → ny + ingine = nyingine

Easy rule: N class vowel adjectives always start with ny!
🍋
JI / MA Class — Fruits & Large Things
JI class uses j- before e/i, or l- before a/o/u.
MA class uses m- before vowel stems (and the vowel blends in).

-ekundu (red) → j + ekundu = jekundu | ma + ekundu = mekundu
-ingine (other) → l + ingine = lingine | ma + ingine = mengine
🧍
M / WA Class (People) & M / MI Class (Trees)
Both use mw- for singular before vowel stems.
WA class plural uses w- | MI class plural uses mi-

-ekundu (red):
Person singular: mwekundu | WA plural: wekundu
Tree singular: mwekundu | MI plural: miekundu

-ingine (other):
Person: mwingine | WA plural: wengine | MI plural: mingine
Full Table — using simple adjectives -ekundu (red) and -ingine (other/another)

Why -A of Association matters HERE
The -a of association (ya, cha, wa, la, vya...) is the SAME set of prefixes used in adjective agreement — especially for exception adjectives in Section C. If you understand the -a connectors, you already know most adjective agreements!

The pattern is: the noun class prefix + the adjective stem. The same prefix that gives you cha in "kitabu cha mwalimu" also gives you ch- in "kitabu chekundu" (the red book).

Same prefix. Same noun class. Just used in two different ways.

Section B
Consonant Stem Adjectives
Adjectives whose stem starts with a consonant — like -zuri (beautiful) or -baya (bad)
💡
The Big Idea
Most adjectives start with consonants — like -zuri (beautiful/good) or -baya (bad). These are simpler than vowel stems because the prefix just sticks on directly without blending.

Each noun class has its own prefix. You match the prefix to the class and you're done!
Noun Class What it covers -zuri (beautiful) -baya (bad) Example
M- sing. 1 person / 1 tree mzuri mbaya mtu mzuria good person
WA- pl. many people wazuri wabaya watu wazurigood people
MI- pl. many trees mizuri mibaya miti mizuribeautiful trees
JI- sing. 1 fruit / big thing zuri baya tunda zuria nice fruit
MA- pl. many fruits mazuri mabaya matunda mazurinice fruits
KI- sing. 1 object kizuri kibaya kiti kizuria nice chair
VI- pl. many objects vizuri vibaya viti vizurinice chairs
N- sing/pl. animals, many nouns nzuri mbaya nyumba nzuria nice house
⚠️ N class has 3 special rules
1
Letters d, g, j, y, z → add n- in front

If the adjective starts with d, g, j, y, or z — add n- before it in N class.

nyumba nzuri ✅  |  -zuri starts with z → nzuri
🐦
ndege ndogo
a small bird
-dogo starts with d → ndogo
🏡

a big house
-kubwa starts with k → NO prefix
🏠

-zuri starts with z → nzuri
🐕

-dogo starts with d → ndogo
🐍

-refu → special form ndefu (N class)
🚗

a nice car
-zuri starts with z → nzuri
2
Letter b → add m- in front

If the adjective starts with b — use m- instead of n- in N class. This is because b and m are both lip sounds — they naturally go together!

nyumba mbaya ✅  |  -baya starts with b → mbaya
🏠

-baya starts with b → mbaya
🛣️

a bad road
-baya starts with b → mbaya
🐈

a bad cat
-baya starts with b → mbaya
3
All other consonants → NO prefix in N class!

If the adjective starts with any other consonant (k, t, p, f, s, h, r, l, w, ch, sh...) — you add NOTHING in N class. The adjective stays exactly as it is!

nyumba kubwa ✅  |  -kubwa starts with k → no prefix, stays kubwa
🏡
nyumba kubwa
a big house
-kubwa starts with k → NO prefix
🐘
tembo kubwa
a big elephant
-kubwa starts with k → NO prefix
🐱
paka hodari
a clever cat
hodari = invariable — stays the same!
🧠 Memory trick for N class rules:
d, g, j, y, z → add n (these letters "hiss" — n makes them softer)
b → add m (b and m are both lip sounds — they match!)
everything else → add nothing (just leave it alone!)
Two adjectives with special forms
📏
-refu (long/tall) → becomes ndefu in N class
Normally you'd expect -refu to become n-refu in N class. But it doesn't — it becomes ndefu instead. Just memorise this one!

meza ndefu = the long table | mti mrefu = a tall tree (M class → mrefu, normal)
✨
-pya (new) has some irregular forms
The adjective -pya (new) is mostly regular but these forms are special:

JI class singular → jipya
N class singular → mpya
N class plural → mpya
U class plural → mpya

kitabu kipya = new book (KI — normal) | nyumba mpya = new house (N — special)
Section C
Exception Adjectives
4 special adjectives that have their own unique rules
💡
The Big Idea
There are 4 adjectives that don't follow the usual rules above. They have their own agreements for each noun class. The good news is that most of their agreements follow the same pattern as the -a of association connectors you already know! (pa, ku, mu, cha, ya, wa etc.)

The 4 adjectives are: -ote (all), -o -ote (any), -enye (having/with), and -enyewe (-self)

Notice how most agreements use the SAME prefix as the -a of association — because it's the same system!

🔑 The shortcut: Look at the -a connector for that noun class. The exception adjective uses almost the SAME prefix!
KI class → -a connector is cha → -ote becomes chote → -enye becomes chenye
VI class → -a connector is vya → -ote becomes vyote → -enye becomes vyenye
N plural → -a connector is za → -ote becomes zote → -enye becomes zenye
Same prefix family — just with a different ending!
1. -ote (all / whole)
👥
watu wote
all people
WA class → wote
📚
vitabu vyote
all books
VI class → vyote
🏠
nyumba zote
all houses
N class plural → zote
🍋
matunda yote
all fruits
MA class → yote
2. -enye (having / with)
-enye must always be followed by a noun or an infinitive verb. It means the noun "has" something or "comes with" something.
💡
kandili yenye mafuta
a lantern with oil
N class → yenye | followed by noun
💰
mtu mwenye pesa
a person with money
M class → mwenye | followed by noun
3. -enyewe (-self)
-enyewe means "himself/herself/itself/themselves." It always comes after the noun or pronoun.
👩
msichana mwenyewe
the girl herself
M class → mwenyewe
🧑
mimi mwenyewe
I myself
used after a pronoun
📋 Word Order with Exception Adjectives
🔢
The Order Rule
When you use multiple adjectives together, this is the order:

Noun → normal adjectives → -ote/-o ote → -enye → -enyewe

Example: Vitabu vikubwa vyekundu vyote = Take all the big red books
(big → red → all — notice -ote comes last)

💬 Exception adjectives in sentences

Askari wote wamerudi.
All the soldiers have returned.
wote = all (WA class)
Mimi mwenyewe sijakwenda.
I myself have not gone.
mwenyewe = myself (after pronoun)
Karoti zote zimeoza.
All the carrots have gone bad.
zote = all (N class plural)
Vitabu vikubwa vyote ni vyekundu.
All the big books are red.
vyote = all (VI class) — comes after adjective vikubwa
Section D
Invariable Adjectives
Adjectives that NEVER change — same for every noun class
🎉
The BEST type of adjective!
These adjectives are mostly from Arabic. They never change at all — it doesn't matter what noun class they're with, they always stay exactly the same. No prefixes, no blending, no rules to memorise. Just use the word as-is! 🎊

mtu safi (a clean person)  |  nyumba safi (a clean house)  |  vitabu safi (clean books)  |  matunda safi (clean fruits)
The word safi stays safi every single time! 🙌
Section E
Compound Adjectives
Adjectives made by joining two words with the -a of association
💡
The Big Idea
Sometimes you need to describe something but there's no single adjective for it. In Swahili, you can build your own description using the -a of association connector (ya, cha, wa, la...) + another word.

Think of it like building with Lego — two words click together to describe the noun! 🧱
Noun being described
+
-a connector ya / cha / la / wa...
+
describing word noun / verb / direction
= compound adjective ✅
💧
maji ya moto
hot water
maji (N) + ya + moto (hot)
📍
mahali pa hatari
dangerous place
mahali (PA) + pa + hatari
📚
chumba cha kusomea
reading room
chumba (KI) + cha + kusomea
👩
bidhaa za kike
women's products
bidhaa (N) + za + kike
🧒
harakati za kitoto
childish activities
harakati (N) + za + kitoto
✂️
makasi ya kunyolea
haircutting scissors
makasi (N) + ya + kunyolea
Fun fact about "bure": On its own, bure as an invariable adjective means "useless." But as a compound adjective, -a bure means "free (no cost)!" For example: Ni bure = It is useless. But Chakula cha bure = Free food! 🍽️
